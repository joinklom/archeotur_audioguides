QR Code — Siti Archeologici

Inserisci il dominio o percorso base dove ospiterai i file HTML, poi clicca Aggiorna QR. Stampa o esporta in PDF per creare i cartelli da esporre nei siti.

Esempio: se i file saranno raggiungibili come https://visita.comune.it/museo.html, inserisci https://visita.comune.it/
Ogni QR punterà automaticamente al file corretto (museo.html, villaggio.html, ecc.)
